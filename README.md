# vanilla-extract-vue 

## Project setup
```
pnpm install
```

### Compiles and hot-reloads for development
```
pnpm run dev
```

### Compiles and minifies for production
```
pnpm run build
```

### Run cypress test suite
```
pnpm run cypress:open
```

### Lint files
```
pnpm run lint
```

### Format files
```
pnpm run format
```
